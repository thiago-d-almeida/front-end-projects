# front-end-projects

Este repositório contém protótipos funcionais desenvolvidos para demonstrar competências em Design Visual, Hierarquia de Informação e Desenvolvimento Front-end. 

## Projetos Inclusos

### 1. Landing Page Jurídica (Almeida Advogados)
* **Foco:** Autoridade, confiança e conversão.
* **Destaque:** Uso de tipografia serifada para autoridade e paleta de cores corporativa (Navy & Gold).
* **Tecnologias:** HTML5, Tailwind CSS.

### 2. Netfish (Netflix Redesign)
* **Foco:** UI/UX e Imersão.
* **Destaque:** Redesign focado em alto contraste e redução de ruído visual para plataformas de streaming.
* **Tecnologias:** HTML5, Tailwind CSS, CSS Interativo.

### 3. ₿ Bitcoin Live Dashboard
* **Foco:** Consumo de dados em tempo real e visualização técnica.
* **Destaque:** Integração dinâmica com API (CoinGecko) e tipografia monoespaçada para legibilidade financeira.
* **Tecnologias:** HTML5, Tailwind CSS, JavaScript (Fetch API).
